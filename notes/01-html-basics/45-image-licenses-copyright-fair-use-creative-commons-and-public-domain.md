# Image Licenses — Copyright, Fair Use, Creative Commons & Public Domain

<aside>
✨

**Big idea:** Images are **intellectual property**. By default they're **"all rights reserved"** — you can't just grab one off Google. Use images only when you have **permission**, a **license**, a **fair use** justification, or the image is in the **public domain** (e.g. **Creative Commons 0**). Stock sites like **Pixabay** and **Unsplash** are safe go-tos.

</aside>

## ⚖️ Images Are Intellectual Property

- Images are protected by **copyright regulations** — ownership usually belongs to the **creator** (sometimes the publisher).
- The **default** license for any image is **all rights reserved** — the copyright holder controls every use of it.

## 🚫 What "All Rights Reserved" Means for You

If an image is all rights reserved, you can use it only by doing **one of three things**:

1. **Get written permission** from the copyright holder.
2. **Purchase a license** from the copyright holder (e.g., Getty, Shutterstock).
3. Use it in a way that qualifies as **fair use**.

## ⚖️ Fair Use

- A legal doctrine allowing limited use of copyrighted material without permission.
- The use must generally be **limited** and **transformative**.
- Common fair use examples:
    - Commentary / criticism / review of the work.
    - Parody.
    - Educational, non-commercial illustrations.
- ⚠️ Fair use is **case-by-case and jurisdiction-dependent** — if in doubt, get permission or pick a different image.

## 🆓 Permissive Licenses (Creative Commons, BSD, MIT)

- Some creators release images under a **permissive license** — you can use them, **but you must follow the license's rules**.
- Common examples:

| License | What it typically requires |
| --- | --- |
| **Creative Commons BY** | Use freely with attribution to the creator |
| **Creative Commons BY-SA** | Attribution + share derivative works under the same license |
| **Creative Commons BY-NC** | Attribution + no commercial use |
| **Creative Commons BY-ND** | Attribution + no modifications (no derivatives) |
| **BSD / MIT** | Usually for code, but can apply to assets — attribution often required |

<aside>
⚠️

Always **read the license** before using a permissive-licensed image. Some require open-sourcing your project, banning modifications, or attributing the creator in a specific way.

</aside>

## 🌐 Public Domain

- A work in the **public domain** has **no copyright** attached.
- You can use it **freely**, **without restrictions** — even commercially, even modified.
- Images licensed under **Creative Commons 0 (CC0)** are considered public domain.
- Other ways an image can enter the public domain: copyright expired, U.S. federal government works, deliberate dedication by the author.

## 🔎 How to Find Images You Can Use

- Use **search engines with license filters** (e.g., Google Images → Tools → Usage rights).
- Use dedicated free-image sites — most images are CC0 or under a permissive license:
    - **Pixabay**
    - **Unsplash**
    - **Pexels**
    - **Wikimedia Commons** (mostly CC-licensed)
- Read each image's license before using it — even on free sites, some images have model release or attribution requirements.

## ✅ Best Practices for Using Images on the Web

- ✅ Default-assume an image is **copyrighted** until you can prove otherwise.
- ✅ Prefer **CC0** or **public-domain** assets for hassle-free use.
- ✅ **Attribute** the creator when the license calls for it.
- ✅ Keep a **record** of where each image came from and under what license — future you (and your legal team) will thank you.
- ❌ Don't assume "no watermark" means "free to use."
- ❌ Don't "credit" your way out of copyright on social-media pulls — that isn't how licensing works.

## ✅ Key Takeaways

- Images are **intellectual property** — by default **all rights reserved**.
- Use a copyrighted image only with **permission**, a **purchased license**, or a valid **fair use** justification.
- **Permissive licenses** (Creative Commons, BSD, MIT) allow use under specific rules — read them.
- **Public domain** (and **CC0**) images are free to use without restrictions.
- For safe, free images, use sites like **Pixabay**, **Unsplash**, and **Pexels** — and still check the license per image.
